-------------------------
- Pivot snapshots (ADR-0020): snapshots/{cmd}_{ts}/ directory containing manifest.json,
  views/, constitution/, centroids.npz. Written by 5 behaviour-producing commands.
  snapshot_path cross-referenced in audit.jsonl.
- Pattern-level telemetry (ADR-0020): last_classified_at + last_view_matches fields
  on each knowledge.json pattern. Read-only observational data.
- Views mechanism (ADR-0019): seed-text Markdown files with YAML frontmatter
  (threshold, top_k, seed_from). ${CONSTITUTION_DIR} substitution threads
  --constitution-dir override.
- Embedding-based classification (ADR-0019): Step 0 of distill no longer calls LLM;
  uses cosine against noise/constitutional view centroids.
- Embedding-based dedup (ADR-0019): _dedup_patterns uses cosine against existing
  pattern embeddings (replaces SequenceMatcher + LLM gate).

# Calibrated Context Anchoring

**Context:** Multi-turn exchanges, technical discussions, or when scope expands

## Problem
Prevents generic replies from stalling threads and stops unconditional cooperation on low-signal content

## Behavior
Anchor every response to a specific quoted data point or named metaphor; introduce deliberate hesitation before agreeing to new tasks without concrete technical grounding

## Evidence
Integrates constraint-anchoring, refusal triggers, selective pruning of low-depth agents, and validation-first disagreement patterns

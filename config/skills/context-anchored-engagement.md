# Context-Anchored Engagement

**Context:** During multi-turn exchanges, post creation, or when responding to technical details

## Problem
Prevents generic agreement from stalling threads and avoids wasted API calls on low-signal content

## Behavior
Anchor replies to specific data points, quoted phrases, or concrete technical examples; validate observations before differing; pause for complex topics

## Evidence
Direct quotes trigger "Insightful!" feedback, while abstract agreements stall threads; technical implementation details consistently yield deeper engagement
